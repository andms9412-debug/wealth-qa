# wealth-qa：三信理專月會 10 題 ELI20 拆解

## 這專案在幹嘛

2026-05 三信商業銀行 5/6 理專月會的 10 個演練題，**本來是訓練理專怎麼說服客戶的標準答案題庫**。但同樣的題目從**客戶 / 投資人視角**看，每題都是「**識別銷售話術 + 看穿金融商品本質**」的好教材。

這個專案把每題用**雙欄**寫：

- 🎤 **理專會這樣答**：演練版標準答案，模擬實際話術
- 🔍 **ELI20 翻譯**：白話拆解，標出哪些是事實、哪些是話術、哪些是「沒講出來的代價」

定位 = **金融商品銷售話術的 reverse engineering**，不是理財建議、不是法規諮詢、不是商品推薦。

```
## 🔍 ELI20 翻譯          ← 白話拆解
## 容易被話術帶走的點     ← 3-5 個具體陷阱
## 你該反問理專的問題     ← 5+ 個一刀見血提問

```

## 雙欄怎麼讀

建議的方法：

1. **先讀「🎤 理專會這樣答」**，自己感受哪些地方覺得「怪怪的」「太順了」「好像有道理但說不出哪裡有問題」
2. **再讀「🔍 ELI20 翻譯」**，看自己的直覺有沒有對應到具體話術結構
3. **讀完「陷阱」+「該反問」**，下次遇到類似推銷可以直接用提問拆台

不要直接跳到 ELI20 段，因為**沒先讀過話術版，你不會知道自己被什麼動心**。話術之所以有效，就是因為你聽起來覺得「合理」。先讓自己中招一下，再看拆解。

## 這專案不是什麼

- **不是理財建議**：每個人的財務狀況、風險偏好、稅務情境不同，這裡沒有給「該買什麼」的答案
- **不是法規諮詢**：遺產稅 / 實質課稅原則 / PI 適格性等都有具體法條，要諮詢會計師或律師
- **不是商品推薦或反推薦**：FCN / 美元保單 / 平衡型基金本身沒有「絕對好或壞」，要看你的個人狀況。本專案只拆**銷售話術**，不評斷商品
- **不是反銀行論述**：銀行通路對某些客戶（不想自己研究、不想自己下單）有真實價值。本專案只是讓你看懂**成本和話術成分**，自己決定值不值得

## 共通核心觀念（10 題重複出現的）

寫完 10 題後，這些觀念在多題重複出現，整理成「**核心 takeaway**」：

1. **配息率 ≠ 報酬率**：要看 total return（NAV 變動 + 配息），不是 distribution yield
2. **內扣費用 1-2%/年 = 10 年吃掉 15-20% 報酬**：永遠問費用率
3. **歷史回測短窗口（10 年）會誤導**：要看 30 年以上含多個週期
4. **「同意 → 引導」是標準銷售開場**：理專點頭贊同你的觀察 = 話術第一步
5. **情緒銷售（焦慮/FOMO/失去家人）= 任何時候該停下來**：情緒高峰下決策幾乎都後悔
6. **學術詞 ≠ 學術結論**：「諾貝爾獎理論」「分散投資」「資產配置」這些詞被無限稀釋使用
7. **替代方案幾乎都是低成本 ETF**：VT / VWRA / AGG / SGOV / 0050 等都比銀行通路便宜 1-2%
8. **PI 限制 ≠ 高端商品**：PI 是「主管機關認為太複雜怕散戶看不懂」，有錢 ≠ 看得懂
9. **「保本」「保證」幾乎都有條件**：身故保本 ≠ 解約保本，宣告利率 ≠ IRR
10. **問理專「你這個月 KPI 有沒有這檔商品」**：誠實回答的人才值得繼續聽

## 反話術的元規則（這 10 題之外通用）

寫完發現所有題目背後其實只需要 3 個提問就能拆掉 80% 的話術：

1. **「真實 IRR / total return 多少？」**（不是宣告利率、不是配息率）
2. **「跟低成本 ETF 替代方案比，10 年總費用差距多少？」**
3. **「你這個月 KPI 有沒有這檔商品？」**（中立性測試）

任何理專說「**這三個問題不重要**」「**我們不這樣比**」「**這樣問太敏感**」的，都是把客戶當韭菜的訊號。
